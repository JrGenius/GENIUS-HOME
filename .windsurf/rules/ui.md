# UI Rules

Every screen must look premium.

Use whitespace.

Large typography.

Rounded corners.

Subtle animations.

Cards instead of containers.

Minimalistic interface.

Do not overload dashboards.

Icons must come from Material Symbols.

Dark mode first.

Responsive.

Tablet friendly.

Desktop friendly.

PWA friendly.

No ugly dialogs.

Prefer Bottom Sheets.

Prefer Snackbars.

Prefer Hero animations.
