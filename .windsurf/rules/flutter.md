# Flutter Rules

Use

Material 3

Riverpod

GoRouter

Freezed

Json Serializable

Drift

Responsive Framework

Never use StatefulWidget unless necessary.

Prefer ConsumerWidget.

Always use const.

Prefer extension methods.

Never use BuildContext inside business logic.

Avoid nested FutureBuilders.

Avoid nested StreamBuilders.

Create reusable widgets.

Use ThemeData.

Never hardcode colors.

Never hardcode paddings.

Never hardcode text styles.

Everything goes through Theme.
